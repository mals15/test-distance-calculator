POST: /calculate/distance/ - метод принимает запрос в формате {
                                                               "calcType": "Distance Matrix",
                                                               "fromCity": "Samara",
                                                               "toCity": "Ulsk"
                                                             }

    calcType - метод расчёта расстояния, может принимать следуюшие значения:
            Crowflight - расчет по прямой по формуле исходя из широты и долготы городов;
            Distance Matrix - расстояние берется из значения в базе;
            All - подсчет расстояния обоими способами;

            ответ на запрос в сделедующем формете в зависимости от выбранного метода:
                                                             {
                                                               "Crowflight": "112",
                                                               "Distance Matrix": "223"
                                                             }
                                                             {
                                                               "Crowflight": "112"
                                                             }
                                                             {
                                                               "Distance Matrix": "112"
                                                             }

GET: /city/all/ - при вызове метода получаем список всех городов в базе в виде:
                                                             {
                                                               "id": "1",
                                                               "name": "Moscow"
                                                             }

POST: /upload/xml - на метод подаем xml файл для загрузки а базу данных пар городов и расстояния между ними;
